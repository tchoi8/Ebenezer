<pre> </pre>
  • Shortened word for 'preformatted text.'
  • Text in a <pre> tag is displayed in a fixed-width font
  • The text preserves both spaces and line breaks.
  • The text will be displayed as exactly as written in the HTML source code. This feature is different from <p> tag which displays the text as a paragraph, and ignores spaces and line breaks.
  • <pre> tag usually contains text with a unique format, code source with indentation format, special characters or ASCII Art. ASCII stands for American Standard Code for Information Interchange, which is a character encoding standard for the web.
  • We are using <pre> tag in order to create a knit based on the student's code. The output of the ASCII design will be printed onto a knit, so the students may touch their work in the end. The knit sample will be sent to students by February 22nd, for the 3rd workshop.
  • Some screen readers cannot properly format pre elements, and may ignore the spacing and line breaks inside of <pre> tag. It is recommended that you provide title or explanatory text along with the <pre> tag.
