Poetic Coding Lesson 1

Basic Structure of HTML Document

Date: February 4, 2021

<html> </html>
  • Shortened word for ‘Hyper Text Markup Language’.
  • Primary language standard used to organize and format web pages and other documents on the World Wide Web. 

<head> Sonia's Castle </head>
  • HTML’s head area. Contains metadata* and set up.
  • First section in the code that contains info about a web page’s properties and links to external related files.
  • *Metadata: data that provides info about other data 

<title> My Little Pony </title>
  • Shows the title of the document.
  • Doesn’t show up on the webpage but on the browser’s toolbar or when you search, it comes up as the title of the webpage. 

<body> Welcome to my Castle </body>
  • Body of the document.
  • Contains all the contents of an HTML document (i.e. headings, paragraphs, images, hyperlinks, etc.) 

<h1> My Universe - Soniaverse </h1>
  • Header
  • Biggest font size for header 

<br> Mr. Twinkles <br/>
  • Inserts a single line break
  • Usually doesn’t require a closing tag, but some programmers like to use the closing tag <br/> for consistency.
